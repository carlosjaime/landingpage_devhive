# DevHive Landing (Next.js)

Landing page profesional para consultora y fábrica de software `DevHive`.

## Incluye

- Hero comercial con métricas y CTA
- Efectos parallax en secciones clave
- Animaciones de entrada por scroll (reveal)
- Secciones completas: servicios, proyectos, metodología, stack, testimonios y contacto
- Diseño responsive para desktop y móvil

## Ejecutar local

```bash
npm install
npm run dev
```

Abre `http://localhost:3000`.

## Build producción

```bash
npm run build
npm start
```

## Despliegue en Vercel

1. Sube esta carpeta (`DevHive`) a un repositorio Git.
2. Importa el repo en Vercel.
3. Framework detectado: `Next.js`.
4. Build command: `npm run build`.
5. Output: `.next` (automático).
